Projects

Samplify

Samplify is an app that allows users to search for any song available on Genius and find the samples used in that song with embedded YouTube players. It has extra features such as an apple music player and links to the lyrics. The app was made using jQuery, Bulma, and 2 third-party APIs, Genius and YouTube.

The Tech Blog

The Tech Blog is an MVC application that allows users to login or create an account, create, update, and delete posts, and add comments to other users posts. The app is deployed to Heroku with JawsDB to store blog posts, comments, and user accounts where the password is hashed with bcrypt. This app was built with jQuery and Bootstrap.

tr-via-hub

tri-via-hub is an MVC application that allows users to create, generate, play, and delete quizzes based on category. It utilises MYSQL/JawsDB to allow users to log in and save their quiz to a database, and uses Handlebars and Sequelize to display the data. Some other featured technologies are jQuery, Bootstrap, and Open Trivia DB, a third-party API to generate questions. The app is deployed to Heroku.

Weather Dashboard

The Weather Dashboard app allows users to search for any city in the world (can also add Country for accuracy) and it will retrieve data from the OpenWeatherMaps API and display the weather details on screen. It also uses local storage to store recently searched cities.
