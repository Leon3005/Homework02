Projects

Samplify

Samplify is an app that allows users to search for any song available on Genius and find the samples used in that song with embedded YouTube players. It has extra features such as an apple music player and links to the lyrics. The app was made using jQuery, Bulma, and 2 third-party APIs, Genius and YouTube.

Weather Dashboard

The Weather Dashboard app allows users to search for any city in the world (can also add Country for accuracy) and it will retrieve data from the OpenWeatherMaps API and display the weather details on screen. It also uses local storage to store recently searched cities.

Horiseon Website Refactor Project

This project required me to refactor the Horiseon website by using semantic HTML elements, fixing links in the nav bar, reducing the amount of declaration blocks in the CSS file, making images on the page accessible by using 'alt' tags, and more.
